# Loading

等待视图

# 基本用法

import { LoadingUtil, Loading, LoadingTitle, Spinner, CircleLoading, GrowLoading } from "react-native-maui";

<Loading />
<LoadingTitle color="white" titleStyle={{ color: 'white' }} title="wait" />
<Spinner activeColor="#ff9900" inactiveColor="#f8e0b0" />
<CircleLoading color="#3E2AD1" />
<GrowLoading />

# 属性

# Loading

名称 描述 类型 默认值
color 颜色 string -
size 大小 small、large -
animating 是否展示动画 boolean -

# LoadingTitle

名称 描述 类型 默认值
color 颜色 string -
size 大小 small、large -
animating 是否展示动画 boolean -
title 展示的文字内容 string -
titleStyle 文字内容样式 TextStyle -

# Spinner

名称 描述 类型 默认值
activeColor 活动状态颜色 string '#1e90ff'
inactiveColor 非活动状态颜色 string 'white'
duration 动画执行持续时间，单位ms number 1000
size 圆点尺寸 number 8
number 圆点个数 number 3

# CircleLoading

名称 描述 类型 默认值
color 圆弧颜色 string '#1e90ff'
duration 动画执行持续时间，单位ms number 1000
circle 圆弧大小，角度[0,360] number 120
size 圆的尺寸 number 30

# GrowLoading

名称 描述 类型 默认值
color 圆弧颜色 string '#1e90ff'
duration 动画执行持续时间，单位ms number 2000
size 圆的尺寸 number 30
